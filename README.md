## ✨ Features

- 🎨 Beautiful and responsive design powered by [Fluid](https://github.com/fluid-dev/hexo-theme-fluid)
- 📱 Mobile-friendly layout
- ⚡ Fast page loading with static site generation
- 🔍 SEO optimized
- 🌙 Dark mode support (Fluid theme feature)
- 💬 Comment system integration
- 📊 Analytics support

## 🚀 Tech Stack

- **Framework**:  [Hexo](https://hexo.io/) - A fast, simple & powerful blog framework
- **Theme**:  [Fluid](https://github.com/fluid-dev/hexo-theme-fluid) - An elegant Material Design theme
- **Hosting**:  GitHub Pages
- **Language**: HTML, CSS, JavaScript

## 📦 Quick Start

- **Create New Post**

```bash
hexo new post "Your Post Title"
```

- **Generate Static Files**

```bash
hexo generate
```

- **Deploy**

```bash
hexo deploy
```

## 📝 Configuration

Edit `_config.yml` to customize your blog settings:

- Site information (title, description, author)
- Theme configuration
- Deployment settings
- Plugin configurations

Refer to [Hexo Documentation](https://hexo.io/docs/) and [Fluid Theme Documentation](https://hexo.fluid-dev.com/docs/) for more details.
